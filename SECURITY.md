# 安全政策

我们非常重视 AMONEY 数字文明创世协议的安全性。我们感谢安全研究人员和社区成员帮助我们识别和修复潜在的漏洞。

## 报告漏洞

如果您发现任何安全漏洞，请**不要**通过公共 GitHub Issue 报告。请通过以下方式负责任地披露漏洞：

*   **电子邮件**：请将详细的漏洞报告发送至 security@amoney.io。

在您的报告中，请包含以下信息：

*   漏洞的详细描述。
*   重现漏洞的步骤（如果适用）。
*   漏洞的潜在影响。
*   任何相关的概念验证 (PoC) 代码或截图。

我们承诺在收到报告后尽快确认，并与您合作解决问题。在漏洞得到修复并公开披露之前，请您不要公开披露漏洞细节。

## 我们的承诺

*   我们将在收到您的报告后 48 小时内确认。
*   我们将努力在合理的时间内修复漏洞。
*   在漏洞修复并公开披露后，我们将感谢您的贡献。

## 范围

本安全政策适用于 AMONEY 数字文明创世协议的所有智能合约和相关代码库。

感谢您帮助我们确保 AMONEY 社区的安全！
